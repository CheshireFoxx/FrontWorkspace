jQuery 개요

jQuery란

        기존에 자바스크립트 언어만으로 구현하기에 복잡했ㄷ던 구문들을 간소화하여 사용할 수 있도록 고안된
        javascript기반 라이브러리
        즉, javascript 방식으로 작성된 다양한 함수 및 메서드들이 존재한다.

        복잡한 DOM 요소 관련 선택 및 조작기능이나, 이벤트 처리기능, AJAX 기능 및 추가 확장 플로그인 등
        다양한 기능을 제공한다.

jQuery 라이브러리 연결 방법

1. head 태그 내부에 script 태그로 jQuery관련 .js파일 가져오기
- jQuery 라이브러리를 사이트에서 직접 다운로드 받는 오프라인 방식
2. CDN(Content Delivery Network)를 통한 다운로드 방식(온라인 방식)
- jQuery사이트에서 제공하는 CDN경로를 src 속성으로 지정.
